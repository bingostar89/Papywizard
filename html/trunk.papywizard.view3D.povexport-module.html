POVEXPORT_VERSION

This module exports a VPython scene as POV-Ray scene description code. Lights and camera location from the current visual scene are included. Optionally, you may specify a list of include files, and pov textures for objects. For an example of its use, see 'povexample.py' convex objects are not exported.
