_driveWithExternalClosedLoop(self, pos)

source code 

External closed-loop drive.

This method implements an external closed-loop regulation. It is faster for angles < 6-7é, because in this case, the head does not accelerate to full speed, but rather stays at very low speed.

Problem: this drive can't be stopped, neither run concurrently on both axis without big modifications in multi-threading stuff.

Parameters:
  • pos (float) - position to reach, in é
